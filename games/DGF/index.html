Creating a player

Using the code below you can create a player GameObject that can move left and right. 
// Create a class for the player
class player extends GameObject {
    MoveLeft(){ // Handle moving to the left
        if(this.Position.locX != 10){  // Stop us from moving too far to the left
            this.Position.locX -= 10;
        }
    }
    MoveRight(){ // Handle moving to the right
        if(this.Position.locX != 940){ // Stop us from moving too far to the right
            this.Position.locX += 10;
        }
    }
}

// Main loop
function GameLogic(){
    EngineLoop();
}

// Add the object to the stack
CreateObject("Player", new player(50, 540, 50, 50, "#FFFFFF", false));
// Select a canvas
selectCan("GameWindow", "#000000");
// Start the engine
var interval = setInterval(GameLogic, 10);

To add the ability for the player to use its movement functions you would add the below code below the creation of the player object.

addInputBind("KeyA", "MoveLeft", "GameObjects['Player']");
addInputBind("KeyD", "MoveRight", "GameObjects['Player']");
